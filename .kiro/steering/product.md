# Product Overview

SIMANIS (Sistem Manajemen Aset Sekolah) is a school asset management system frontend application.

## Core Features

- **Asset Management**: CRUD operations for school assets with QR code support
- **Inventory**: Stock opname with QR scanner functionality
- **Loans**: Asset borrowing/lending management
- **Dashboard**: Statistics and recent activity overview
- **Reports**: Generate KIB reports (Excel/PDF format)
- **Audit Trail**: Track data changes and modifications
- **PWA Support**: Installable and offline-capable progressive web app

## Target Users

School administrators and staff managing physical assets, inventory, and equipment.

## Key Characteristics

- Mobile-first responsive design
- Supports both mock API (development) and real backend
- Indonesian language interface
- Comprehensive logging and error handling
- Offline capability through PWA
